# Wi-Fi-laser-with-LCD-I2C-and-2-arduino


 Board I2C / TWI pins
- Uno       =   A4 (SDA), A5 (SCL)
- Mega2560  =   20 (SDA), 21 (SCL)
-Leonardo  =   2 (SDA), 3 (SCL)
- Due       =   20 (SDA), 21 (SCL), SDA1, SCL1


##LISTA DE MATERIALES

- 2 ARDUINO UNO
- 1 LCD I2C
- 1 Potenciometro 2k (colocar la resistencia variable en 1.3k)
- Varios cables
- 1 ldr
- 1 laser color verde
- 1 tip131
- 2 resistencia de 220
- 1 led verde
